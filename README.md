# PetroTrans AI — Bản dùng thử (beta)

Webapp dịch thuật đa ngôn ngữ (Anh · Trung · Nhật · Nga ↔ Việt) chuyên ngành Dầu khí - Hóa chất & Xuất nhập khẩu.

- Dùng thử: mở trang web, vào tab "⚙ Cài đặt API", dán API key (lấy miễn phí tại https://aistudio.google.com/apikey), chọn ngành và dịch.
- Dữ liệu (key, từ điển, bộ nhớ dịch) chỉ lưu trên trình duyệt của bạn — không có máy chủ thu thập.
- Bản dịch AI cần người có chuyên môn duyệt trước khi dùng chính thức/pháp lý.
